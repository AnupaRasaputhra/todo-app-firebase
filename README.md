# todo-app-firebase

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository OR Download ZIP

    ```bash
        git clone <repository_url>
    ```

2. Navigate to the project directory

    ```bash
        cd todo-app-firebase
    ```

3. Install dependencies

    ```bash
        npm install
    ```

4. Run the project

    ```bash
        npm start
    ```
